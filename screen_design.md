# 営業日報システム 画面定義書（シンプル版）

## 1. 画面一覧

| 画面ID | 画面名             | 説明                 |
| ------ | ------------------ | -------------------- |
| SC-01  | ログイン画面       | システムへのログイン |
| SC-02  | 日報一覧画面       | 日報の一覧表示       |
| SC-03  | 日報登録・編集画面 | 日報の新規登録・編集 |

---

## 2. 画面詳細定義

### SC-01: ログイン画面

#### 画面概要

システムへのログイン機能を提供する。

#### 表示項目

| 項目名         | 入力形式   | 必須 | 説明                       |
| -------------- | ---------- | ---- | -------------------------- |
| メールアドレス | テキスト   | ○    | 営業担当者のメールアドレス |
| パスワード     | パスワード | ○    | ログインパスワード         |

#### ボタン

- **ログイン**: 認証処理を実行してシステムにログイン

#### 画面遷移

- ログイン成功 → SC-02（日報一覧画面）

---

### SC-02: 日報一覧画面

#### 画面概要

自分が登録した日報を一覧表示する。

#### 一覧表示項目

| 項目名   | 説明               |
| -------- | ------------------ |
| 日報日付 | 日報の対象日       |
| 訪問件数 | その日の訪問記録数 |
| 更新日時 | 最終更新日時       |

#### ボタン

- **新規登録**: 日報登録画面へ遷移
- **編集**: 各行の編集ボタンで日報編集画面へ遷移

#### 画面遷移

- 新規登録ボタン → SC-03（日報登録・編集画面）
- 編集ボタン → SC-03（日報登録・編集画面）

---

### SC-03: 日報登録・編集画面

#### 画面概要

営業担当者が日報を登録・編集する。

#### 入力項目

##### 基本情報

| 項目名   | 入力形式 | 必須 | 説明                             |
| -------- | -------- | ---- | -------------------------------- |
| 日報日付 | 日付     | ○    | 日報の対象日（デフォルト: 今日） |

##### 訪問記録（複数行入力可能）

| 項目名   | 入力形式       | 必須 | 説明                   |
| -------- | -------------- | ---- | ---------------------- |
| 顧客名   | テキスト       | ○    | 訪問した顧客の会社名   |
| 訪問時刻 | 時刻           | ○    | 訪問した時刻           |
| 訪問内容 | テキストエリア | ○    | 訪問時の商談内容や状況 |

##### 課題・計画

| 項目名                    | 入力形式       | 必須 | 説明                 |
| ------------------------- | -------------- | ---- | -------------------- |
| 課題・相談事項（Problem） | テキストエリア | -    | 今日の課題や相談事項 |
| 明日の計画（Plan）        | テキストエリア | -    | 翌日の業務計画       |

#### ボタン

- **訪問記録を追加**: 訪問記録の入力行を追加
- **削除**: 各訪問記録の削除ボタン
- **保存**: 日報を保存
- **キャンセル**: 入力内容を破棄して一覧画面へ戻る

#### 画面遷移

- 保存後 → SC-02（日報一覧画面）
- キャンセル → SC-02（日報一覧画面）

#### バリデーション

- 訪問記録は最低1件必須
- 顧客名・訪問時刻・訪問内容は必須入力

## 4. 補足事項

### 共通仕様

- 各画面には共通ヘッダーを配置（ログインユーザー名、ログアウトボタン）
- エラーメッセージは画面上部に赤字で表示
- 成功メッセージは画面上部に緑字で表示
- 必須項目には項目名の横に「\*」を表示

### シンプル化のポイント

- 顧客マスタは省略し、日報登録時に直接顧客名を入力
- 営業マスタは省略し、ログイン機能のみ実装
- 上長コメント機能は省略
- 検索・絞り込み機能は省略し、一覧表示のみ
